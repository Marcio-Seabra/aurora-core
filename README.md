# aurora-core
Aurora Core é o núcleo de uma IA administradora com memória externa. O sistema ingere conversas, divide e classifica memórias (identity/short/long), aplica deduplicação e sanitização contra prompt‑injection, monta contexto e roteia a resposta. Funciona com LLM local via Ollama (modelos Aurora baseados em Llama 3.1) e suporta modos de desempenho.
